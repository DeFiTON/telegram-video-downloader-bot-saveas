# Features

## Multi-platform downloads

Download videos from YouTube (including Shorts), TikTok, Instagram and Twitter/X.

## High-quality downloads

Videos are downloaded in good quality where the source allows.

## Simple workflow

No interface to learn — send a link, get the video file in chat.

## Multilingual interface

The bot offers a friendly, easy-to-use experience.

## Runs inside Telegram

Everything happens in chat — there is no separate app or website to use.

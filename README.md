## What it is

**SaveAsVideoFree** is a free **Telegram video downloader bot**. Send it a link and it downloads the video and returns the file in chat — no website, no app, no hassle.

It is built for anyone who wants a fast, reliable way to **save videos from social media** straight from Telegram.

## Supported platforms

- **YouTube** — videos and Shorts in high quality
- **TikTok** — fast, watermark-free downloads
- **Instagram** — posts and reels
- **Twitter / X** — video content

## Key features

- **Multi-platform downloads** — YouTube, TikTok, Instagram and Twitter/X.
- **High quality** — videos downloaded in good quality where available.
- **Simple workflow** — send a link, receive the file.
- **Multilingual** — friendly, easy-to-use interface.
- **Runs inside Telegram** — nothing to install.

## How to use

1. Open the bot: [@SaveAsVideoFreebot](https://t.me/SaveAsVideoFreebot)
2. Press **Start**.
3. Send a link to a video from a supported platform.
4. The bot downloads it and sends the video file back to you.
